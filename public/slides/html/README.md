## 🎬 사용 방법

### 슬라이드 시작
1. `index.html` 파일을 브라우저에서 열기
2. "프레젠테이션 시작 🚀" 버튼 클릭
3. 키보드나 클릭으로 네비게이션

### 키보드 단축키
- `→` 또는 `스페이스바`: 다음 슬라이드
- `←`: 이전 슬라이드
- `Home`: 첫 번째 슬라이드로
- `End`: 마지막 슬라이드로

### 투표 슬라이드
- 옵션 클릭 또는 숫자키(1-4) 사용
- 투표 완료 후 다음 슬라이드 이동 가능
