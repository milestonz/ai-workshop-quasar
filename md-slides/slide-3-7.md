# ➄ 예시 제공

- Few-shot Prompting
  원하는 결과물의 형식이나 스타일을 명확히 전달하기 위해, 한두 가지 좋은 예시(Few-shot)를 함께 제공하는 기법입니다.

❌ 나쁜 예

```
교회 소식지 제목을 만들어줘.
```

✅ 좋은 예

```
아래 예시처럼, 교회 소식지에 사용할 따뜻하고 감성적인 제목 3가지를 만들어줘.
예시 1: '사랑의 빛, 세상을 밝히다'
예시 2: '은혜의 강가에서, 쉼을 얻다'
```

💡 핵심 고려사항
예시는 최종 결과물의 '견본' 역할을 하므로, 원하는 결과물의 방향과 가장 유사한 예시를 제시하는 것이 중요합니다.
